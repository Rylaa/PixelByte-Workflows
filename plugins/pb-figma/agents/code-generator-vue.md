# Vue Code Generator (Coming Soon)

This agent is planned for v1.2.0.

## Planned Features

- Vue 3 Composition API with `<script setup>`
- TypeScript support with defineProps and defineEmits
- Tailwind CSS integration
- Composables for reusable logic (useDesignTokens, useAccessibility)
- Proper ref/reactive usage following Vue 3 best practices
- Dark mode with CSS variables
- ARIA attributes for accessibility
- Component organization following Vue style guide

## What Works Now

Use `code-generator-react` for immediate Figma-to-code conversion.
React components can be converted to Vue manually.
